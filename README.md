# QR Code Generator

A polished and modern web-based QR code generator built with HTML, CSS, and JavaScript. This version is designed to feel clean, premium, and effortless while giving users a fast way to create high-quality QR codes for links, text, and everyday sharing.

## 1. Overview

This project is a single-page QR generator that runs directly in the browser. It was upgraded to offer a more refined experience with a sleek interface, better visual presentation, and practical actions such as copy, share, print, and download.

The current implementation in the app reflects a more complete and user-friendly version of the generator, with features that make QR creation feel smooth and professional.

## 2. What’s New

The updated version introduces several improvements over the basic QR generator:

- A more polished and modern visual design
- Cleaner QR presentation with a refined card-style layout
- A centered logo option so users can place a brand or image in the middle of the QR code
- Multiple size options such as Small, Medium, and Large
- Copy, share, and print actions for quick usage
- A Recent section to revisit previously generated QR codes
- A simple experience that works without login or account setup

## 3. Key Features

### QR Generation

- Generate QR codes from any URL or text instantly
- Create scannable QR codes with a clean and neat appearance
- Support for different output sizes for different use cases

### Custom Design

- Add a logo in the center of the QR code
- Keep the QR visually polished while remaining functional
- Enjoy a refined interface with smooth animations and modern styling

### Quick Actions

- Download the QR code as an image
- Copy the generated QR code quickly
- Share it directly through supported browser features
- Print the QR code when needed

### Recent History

- Save and display recently generated QR codes
- Reuse previous QR creations without generating them again
- Clear the recent list anytime

## 4. How It Works

1. Open the app in your browser.
2. Paste a URL or enter text you want to convert into a QR code.
3. Choose the preferred size for the QR output.
4. Optionally upload a logo to place in the center of the QR code.
5. Click Generate to create the QR code.
6. Use the available actions to download, copy, share, or print the result.

## 5. How the Logo Feature Works

The logo feature allows users to add a custom image in the center of the QR code. The app reserves a clean central area for the logo so the QR stays readable and scannable while still looking visually attractive.

This makes it useful for:

- Brand identity
- Business cards
- Event promotions
- Product packaging
- Personal projects

## 6. How the Size Feature Works

Users can select QR sizes depending on their need:

- Small: suitable for compact use
- Medium: balanced for general use
- Large: ideal for print-ready outputs and bigger displays

This gives users flexibility to create QR codes that fit different platforms and visual requirements.

## 7. How the Recent Feature Works

The Recent section keeps track of the latest generated QR codes. Once a QR is created, it is stored in the browser so the user can quickly access it later without generating it again.

This feature is helpful for:

- Reusing recent designs
- Comparing generated QR codes
- Saving time during repeated work

## 8. No Login Required

One of the best parts of this app is that it does not require login or account creation.

Everything works directly in the browser:

- No sign-in required
- No backend account system needed
- No personal data is required to generate a QR code
- Recent items are stored locally in the browser for convenience

This makes the tool fast, lightweight, and accessible for anyone who wants to generate a QR code instantly.

## 9. Technology Stack

- HTML5 for structure
- CSS3 for styling and polished UI design
- JavaScript for QR generation and interactive features
