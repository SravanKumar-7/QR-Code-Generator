# QR Code Generator

A modern and simple web-based QR code generator built with HTML, CSS, and JavaScript. This project lets users paste any link and instantly generate a QR code that can be downloaded for personal or professional use.

## Project Description

This project was developed as a clean and user-friendly tool for generating QR codes quickly and easily. It is designed for smooth use on both desktop and mobile devices and provides a polished experience with a minimal interface.

## Features

- Paste any URL or text and generate a QR code instantly
- Clean and modern dark-themed user interface
- Smooth animations and interactive UI elements
- Download generated QR code as an image
- Works directly in the browser without requiring any installation
- Built as a lightweight single-page web app

## How It Works

1. Open the website in your browser.
2. Enter the link or text you want to convert into a QR code.
3. Click the Generate button.
4. View the QR code and download it when ready.
